ARI NLP Tools

An open source NLP engine for facilitating and improving interactive services with GP call queues

Source Code Watch Demo Video

Abstract

The ARI NLP Tool is an innovative solution aiming to facilitate and improve both front-end and back-end users' interactive experience with the current GP surgery call queues. Together with team 34's "ARI" graphics project, we aim to deliver an easier and more smooth process for patients in need by analysing audio call questions that take place when patients call into GP clinics and direct the them to the corresponding place as quick and accurate as possible.

Apart from improving patients' experience, the solution also provide user friendly interface that allows back-end users (e.g. receptionist) to handle calls very easily. The project also provides a clear visualisation of data of incoming calls and allows researchers to further analyse when needed.

Features

The ARI NLP tool was designed based on numerous interviews with clients and users to maximise both front-end and back-end user experience.

Available offline

Due to the GDPR regulations, we make sure the solution is running offline to best protect our users' privacy

Speech-to-text engine

We tested 4 different speech-to-test solution and picked the most efficient one to ensure both accuracy and efficiency of the project

User interface

The user interface of our proejct is straight forward and easy to use - aiming to deliver the simplist way to configure telephone numbers and IVR systems. A statistical analysis page is also featured to provide a clear visualisation of call status.

Docker solution

The use of docker container allows our solution to sit on different operating systems, namely Linux, Ubuntu and Windows
