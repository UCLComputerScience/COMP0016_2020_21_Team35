GP IVR

An open source Interactive Voice Response (IVR) system for facilitating and improving interactive services with GP call queues

"This excellent project has gone beyond the original scope, and also enables any organisation involved in health and social care to deploy an IVR simply." — Dr Joseph Connor, NHS

Source Code Watch Demo Video

Abstract

Calls to GP surgeries have increased by 200% due to Covid19. The GP IVR is an innovative solution aiming to improve GP call queues. We do this by redirecting a user to a relevant clinician, rather than all patients going through a GP receptionist.

We deliver a better experience for patients by taking them through an initial triage of questions before redirecting them to the right clinician, depending on their answers. Our system gathers yes/no answers from a caller, using open source speech to text run locally. For this reason, our solution is fully compliant with GDPR.

Apart from improving patients’ experience, the solution also provides a user-friendly interface that allows back-end users (e.g. receptionist) to create IVR call systems and handle calls with ease. The GP IVR tool also provides clear visualisation of incoming call data, allowing for further analysis and changes to GP staff structures.

Features

The GP IVR was designed based on numerous interviews with clients and users to maximise both front-end and back-end user experience.

Available offline

Due to the GDPR regulations, we make sure the solution is running offline to best protect our users' privacy

Speech-to-text engine

We tested 4 different speech-to-test solution and picked the most efficient one to ensure both accuracy and efficiency of the project

User interface

The user interface of our proejct is straight forward and easy to use - aiming to deliver the simplist way to configure telephone numbers and IVR systems. A statistical analysis page is also featured to provide a clear visualisation of call status.

Docker solution

The use of docker container allows our solution to sit on different operating systems, namely Linux, Ubuntu and Windows
